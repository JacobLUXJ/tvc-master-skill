# TVC Master Skill
 **全流程TVC智能执行协同系统**
 ---
 ## Overview
 百万级商业TVC制作全流程标准化执行工具。基于12个核心岗位角色打造的「单Agent多模块协同体」，实现从创意到成片的全流程自动
 化管控。
 ```
 总控中枢 → 4大单元 → 12个角色原子Skill
 ```
 ---
 ## Architecture

 | Level | Module | Description |
 |-------|--------|-------------|
 | **L1** | 总控Skill | 全局流程引擎·规则管控·数据中枢·异常兜底 |
 | **L2** | 单元子Skill×4 | 创意策划 / 拍摄执行 / 造型视觉 / 后期制作 |
 | **L3** | 角色原子Skill×12 | CD·文案·AD·分镜师 / 导演·DP·灯光·现场美术 / 服装·化妆·发型 / 剪辑·调色·特效 |
